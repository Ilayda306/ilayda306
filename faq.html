FAQ

Frequently asked questions

1. Hoe kan je een document printen vanuit je laptop op HZ?
HZ heeft publieke printers waar wij als student gebruik van kunnen maken. Dit doe je door je HZ pas te laten lezen via de kaartlezer aan de rechterkant van de printer. Hierna kun je vanaf het display kiezen voor Print Release om te printen.

2. Hoe scan je een document en stuur je het naar je laptop op HZ?
Hz heeft publieke printers waar wij als student gebruik van kunnen maken. Dit doe je door je HZ pas te laten lezen via de kaartlezer aan de rechterkant van de printer. Hierna kun je vanaf het display kiezen voor Scan om documenten te scannen en te versturen naar je HZ mailbox.

3. Hoe kan ik iets kopen (zoals het inschrijven voor de IT introductie activiteit) op de HZ web shop?
Via de HZ web shop kun je HZ producten (zoals een HZ canvas tas) bestellen, maar ook kun je de Cambridge cursussen vinden en je inschrijven voor introductiedagen.

4. Hoe reserveer je ruimte in een van de wings?
Via de HZ University Home page kun je naar Selfservice Portal. Vanuit hier kan je naar "Maak een reservering" en vervolgens "Reserveer een ruimte". Nu alleen nog een vrij lokaal zoeken en voila!

5. wat zijn de instructies als je jouw auto wilt parkeren in de HZ parkeer plaats?
Gratis parkeren is mogelijk op het parkeerterrein van de Poelendaelesingel 12 in Midderlburg, wat op een paar minuten loop afstand is naar de HZ. Het is niet mogelijk om direct bij HZ-pand te parkeren, maar er is wel een invalideplaats beschikbaar direct naast het HZ-pand aan Het Groene Woud1-3. Een HZ-pas is verplicht om het terrein te betreden.
